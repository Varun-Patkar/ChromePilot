**About Section:**
```
ChromePilot is a Chrome extension that brings AI vision capabilities to your browser. Using Ollama's qwen3-vl model running locally on your machine, it can see and understand any web page through screenshots and HTML analysis. Ask questions, get explanations, and receive guidance - all while maintaining complete privacy. v1.0 is a read-only assistant focused on understanding and advising, with no automated actions or browser control.
```
